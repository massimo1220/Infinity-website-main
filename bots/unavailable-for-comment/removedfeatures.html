Feature Graveyard

Sadly, some features always have to be removed. Whether they no longer work, or there's additional complications, there'll always be some.

Music

19/08/20 - 30/12/2020 and ~August 2021

Following a new host, music was originally removed temporarily. Then, it was reinstated for roughly 12 hours, alas the takedown of Groovy ensued, which spelled the end for YouTube music bots. Music was removed and will not be reinstated.

Ranking

03/08/20 - 22/09/21

Following Discord bot verification, we were not granted access to the Privileged Intent required to calculate how long a user spent online, hence we have been forced to end the feature. 

The following commands will no longer collect data, although will still be visible: 
- onlinegraph 

The following commands will run excluding this data: 
- rank 
- leaderboardgraph (lbgraph) 
- leaderboard (lb)
