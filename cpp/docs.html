Writing your first program

For reference, // denotes a single-line comment in C++.

#include <iostream> // Importing this allows us to take input and output to the console

using namespace std; // Without this, you'd have to write std::cout . In more complicated programs which use many libraries, this may prove troublesome if libraries have functions with the same name, which is when we wouldn't want to use this line of code.

int main() // When we compile and run the program, the code within this function executes.
{
    cout<<"Hello world!"<<endl; // This outputs "Hello World!" into the console window when you run the program. endl denotes a new line after printing the hello world.

    return 0; // Returns an exit status of 0 (success). Most compilers will automatically add this line if not included.
}

Integers and Arithmetic

An integer is a variable type (a variable is effectively a way to store data).

#include <iostream>

using namespace std; 

int main() // When we compile and run the program, the code within this function executes.
{
    int myInt1; // This declares an integer. Effectively, we told the compiler that we want this variable to exist, but we haven't assigned a value to it yet.
    myInt1 = 40; // When we first assign a value to an integer, this is called initialisation. Here, we have set the value of the integer variable, myInt1, to 40.

    int myInt2 = 31; // This declares and initialises an integer value in a single line of code, effectively the same as the two lines above, except of course we've set this integer to 31 not 40.

    cout<<"Value 1: "<<myInt1<<endl; // Output the contents of myInt1 into the console
    cout<<"Value 2: "<<myInt2<<endl; // Output the contents of myInt2 into the console

    cout<<"Multiplied: "<<myInt1*myInt2<<endl; // outputs the numbers multiplied together. This could also be written similar to the line below
    int divided = myInt1/myInt2; // Divide myInt1 by myInt2. Note, because they are stored in an integer, the resulting number will be rounded
    cout<<"Divided: "<<divided<<endl; // Output the contents of the variable divided
    cout<<"Added: "<<myInt1+myInt2<<endl;
    cout<<"Subtracted: "<<myInt1-myInt2<<endl;

    return 0; // Returns an exit status of 0 (success). Most compilers will automatically add this line if not included.
}

Strings and For Loops

There are many ways to write code to iterate through the contents of a string. Here, we cover 3 methods.

Option 1 (using substrings)

#include <iostream>
#include <string>

using namespace std;

/*
Notes on string.length() and why i < myString.length() not i <= length():

One thing to note, as that when we're getting the position of a character, substring, element of an array etc, the computer starts counting from 0.
Hence, here's what the string "hello" would look like to the computer:
Index / Position (i): 0 1 2 3 4
Character:              h e l l o

However, notice that this string has 5 letters, and the last letter is at position 4?
That's why we use < not <= in the scenario below, because if we used <=, on the last iteration of the for loop, the computer would try to find position 5 (which doesn't exist), and would error / cause unexpected results.
*/

int main()
{
    string myString = "hello world";

    for (int i = 0; i < myString.length(); i++) // Set integer i to zero at the start (i referencing index, the position of the element in the string that we're checking); Iterate while i is smaller than the length (note you cannot use <=, see comment above); Every iteration (every tine the code in the loop runs once), we increment i (add 1 to i)
    {
        cout<<myString.substr(i, 1)<<endl; // Prints every character on a new line
        // above, the 1 signifies that we're selecting 1 character, and this is returned to the code. For instance, if we used 2, the substr would return 2 Characters
        // note that substr returns a string, not a character variable type
    }
    
    return 0;
}

Option 2 (uses subscription)

#include <iostream>
#include <string>

using namespace std;

int main()
{
    string myString = "hello world";

    for (int i = 0; i < myString.length(); i++) // Set integer i to zero at the start (i referencing index, the position of the element in the string that we're checking); Iterate while i is smaller than the length (note you cannot use <=, see comment above); Every iteration (every tine the code in the loop runs once), we increment i (add 1 to i)
    {
        cout<<myString[i]<<endl; // Prints every character on a new line (here the [i] signafies that we're selecting a character from position i)
        // note that this returns a variable of type character, NOT a string. This can be useful when used in conjunction with the ctypes.h library
    }

    return 0;
}

Option 3 (iterate without i)

#include <iostream>
#include <string>

using namespace std;

/*
Only use this option when we don't care about the variable i, in a case where we would have only used i to extract the character.
*/

int main()
{
    string myString = "hello world";

    for (char &myCharacter: myString) // you may find this easier to read than option 2.
    {
        cout<<myCharacter<<endl; // Prints every character on a new line
    }

    return 0;
}

#include <iostream>

using namespace std;

/*
A function signature is structured like below:
<return-type> functionName(variableType parameter)
ie.
*/

int doubleNumber(int inputNumber) // note, parameters must be declared like so in the function signature
{
    return inputNumber * 2;
}

string duplicateString(string inputString)
{
    return inputString + inputString; // concatenate the two strings together, ie. with inputString set to "hello", this function  would return "hellohello"
}

/*
Default parameters:
These are declared in the function signature, ie.
*/

int multiplyNumber(int inputNumber, int multiplyBy = 2) // here, multipyBy is set to 2 unless otherwise specified
{
    return inputNumber * multiplyBy;
}

int main()
{
    // For instance, we can call the function with and without specifying multiplyBy:
    cout<<multiplyNumber(3)<<endl; // this will use the default parameter to multiple
    cout<<multiplyNumber(3, 3)<<endl; // this overrides the default multiplier of 2, instead setting multiplyBy to 3.
    return 0;
}

Vector Arrays

#include <iostream>
#include <vector>
#include <string>
#include "modules/files.hpp" // Importing a modified version of a file handler I wrote for above code blocks from the same folder. This imports functions which allow me to read and write to files.

using namespace std;

vector<string> splitByChar(string inputString, char splitChar = ',') // Function that takes in a string, ie. "a,b,c,d", and outputs the contents in a vector array, ie (a,b,c,d). The character to split by here is set to ',' by default.
{
    vector<string> splitString;
    string currentItem; // the string that will hold each individual item while being split, then empty and move onto the next item.

    for (const char &character: inputString) // Iterate through every character in the string. It can be considered bad practice to change a letter in a string as you iterate through it, so here we keep it constant.
    {
        if (character == splitChar)
        {
            splitString.push_back(currentItem); // Place the currentItem at the end of the vector array
            currentItem = ""; // Reset this to empty, and we can start appending characters to it again
        }
        else
        {
            currentItem += character;
        }
    }

    return splitString;
}

int main()
{
    string fileContents = readFile("data/names.csv"); // Open the csv file, and save the whole file into one string called fileContents. This csv file is a list of names.
    cout<<"Here's what the file looked like when we opened it:"<<endl<<fileContents<<endl; // Output the whole file's contents to the console
    cout<<"But if we split it into a vector array, and then output each element line by line:"<<endl;
    for (string &name: splitByChar(fileContents)) // Split the list of names from fileContents, by a ',' which places them into a vector array. Then output each element to the 
    {
        cout<<name<<endl;
    }

    return 0;
}

Reading and writing to files

#include <iostream>
#include <string>
#include <fstream> // This library is required for reading and writing to files (file streams)

using namespace std;

string readFile(string filename)
{
    ifstream fileObject(filename); // Here, we initialize a file object. Effectively, we've setup an object to allow reading of the file, which we can then use like below
    string fileContents; // Declares a string to save the contents of the file to.
    string lineOfFile; // Declares a string which we'll save each line of the file to, before appending to the string fileContents.

    while (getline(fileObject, lineOfFile)) // Iterate through every line in the file
    {
        fileContents += lineOfFile + "\n"; // Add each line of the file, to the string that we're writing its contents to, then add a newline in the string.
    }

    fileObject.close(); // Close the file object, and disassociate from the file stream

    // However using the method shown above, we get an extra newline at the end of the file. To solve this, we exclude the last character, which is this newline:
    if (fileContents.length() > 1)
    {
        fileContents = fileContents.substr(0, fileContents.length()-1); // exclude the last character (newline)
    }

    return fileContents;
}

void writeToFile(string filename, string contents)
{
    ofstream fileObject(filename); // Here, we initialize a file object. Effectively, we've setup an object to allow writing to the file, which we can then use like below

    fileObject<<contents; // Write the contents to the file
    fileObject.close(); // Close the file object, and disassociate from the file stream
}

int main()
{
    cout<<readFile("input.txt");
    writeToFile("output.txt","Hello \nWorld!");
}
